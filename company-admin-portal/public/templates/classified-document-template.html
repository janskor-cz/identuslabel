How to Use This Template

This template demonstrates how to create documents with section-level security classification.

Clearance Levels (in order of sensitivity):

  • UNCLASSIFIED - Public information, no restrictions
  • CONFIDENTIAL - Sensitive business information
  • SECRET - Highly restricted information
  • TOP_SECRET - Classified information (highest restriction)

How to Mark Content:

  • Add data-clearance="LEVEL" attribute to any HTML element
  • Works on: <section>, <div>, <p>, <span>, <table>, <tr>, etc.
  • Content without data-clearance is treated as UNCLASSIFIED
